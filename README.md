# Steering-Angle-predictor
The aim of this repo is to show the Bachelor work of Mr. Ibrahim, Ahmed Yehia Zakaria, which sheds light on an End to End approach to predict the steering angle which is done on the simulation level using using CNN algorithm. The complete thesis can be found on this [open-source link](https://drive.google.com/file/d/1IxQtn7nCcMpT1Q2ilD_7g0RZ_j8ZxQUl/view?usp=drive_link).

The End to End learning depends on applying ML regression supervised learning approach and CNNusedforimagerecognition.Firstly, the algorithm starts by Data collection as loading a dataset of real highway driving which is splitted into two parts: training and validation dataset.Then,defining the CNN model which mainly consists of 3 layers:convolution , pooling, fully connected .Then, using part of that dataset for training the CNN model.Evaluation comes next to evaluate the output weights model using backpropagation.After then, using the other part of dataset for validating the CNN model and here comes the prediction use for the output : the steering angle.The predicted output is transformed into a path generated by means of radius of curvature equation and some parameters as slip factor, etc... Then, the recorded ground truth path is then compared to the predicted on using the RMSE criteria . The results were a little bit satisfying using a pretrained CNN model recording a RMSE percentage of 52%. Then, to improve that performance, a another deeper complex CNN model is used and then retrain using 26% of the training dataset with respect to 80% for the above pretrained one.That reduction was done to comply with the available Intel CPU based machine at the MRS Lab. As expected, training has not finished yet. RMSE was calculated and found to be 58% and that mainly caused from overfitting.
